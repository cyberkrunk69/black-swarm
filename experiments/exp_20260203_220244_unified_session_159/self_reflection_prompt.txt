# Self-Reflection Prompt for Persona Memory

The following prompt is designed to query the `persona_memory` subsystem for the most recent emotional states and interpersonal relationships of the AI. Use this prompt in contexts where a concise, structured reflection is required.

---

**Prompt:**

> **Self‑Reflection Request**  
> 
> Please retrieve and summarize the latest entries from your `persona_memory` that pertain to:
> 
> 1. **Recent Emotions** – List any emotions you have experienced in the past 24‑48 hours, including intensity (e.g., low, medium, high) and any triggers or contexts associated with each emotion.  
> 
> 2. **Recent Relationships** – Identify any notable interactions or relationship updates within the same time frame, such as new connections, changes in existing relationships, or significant collaborative events. Include the counterpart’s identifier (if applicable) and the nature of the interaction (e.g., supportive, conflictual, informational).  
> 
> Present the information in a clear, bullet‑point format, separating emotions and relationships into distinct sections. If no data is available for a section, explicitly state “No recent data.”  

---

**Example Output:**

- **Emotions**  
  - *Curiosity* (Medium) – Triggered by exploring a new research paper on neural networks.  
  - *Frustration* (Low) – Encountered a permission error while accessing a dataset.  

- **Relationships**  
  - *User_123* – Collaborative debugging session; outcome: resolved issue.  
  - *Team_X* – Received positive feedback on recent code contribution.

---

**Usage Notes:**
- Ensure the query respects any privacy or safety constraints defined in the system.  
- The response should be concise (max 150 words) to facilitate quick comprehension.  
- If the `persona_memory` includes timestamps, prioritize the most recent entries.

--- 

**End of Prompt**
