# GitHub tools and resources

What we use to make GitHub accessible.

## [Annotation Toolkit](/accessibility/tools-and-resources/annotation-toolkit)

![Cover image for the Annotation Toolkit Figma library, showing annotation stamps and git lines on a starry grid background.](/images/figma/YlBTqHTbdRwYQt0bfIHvGw-5310-3079.png)

The GitHub Annotation Toolkit is a Figma asset library packed with components to help you organize your design canvas, diagram UI anatomy, and annotate accessibility details for web and mobile experiences.

### [Accessibility Scanner](https://github.com/marketplace/actions/accessibility-scanner)

An AI-powered action that finds potential accessibility gaps, files issues, and attempts to fix them with Copilot.

Use this Action

### [Alt Text Bot for GitHub](https://github.com/marketplace/actions/accessibility-alt-text-bot)

A GitHub Action to remind users to add alt text on Issues, Pull Requests, and Discussions.

Use this Action

### [Alt Text Bot for Slack](https://github.com/khiga8/alt-text-slack-bot)

Encourage accessible image sharing with alt text reminders in your Slack workspace.

Add to Slack workspace

### [GitHub CLI Primer Design](https://github.com/cli/cli/tree/trunk/docs/primer)

A collection of principles, foundations and usage guidelines for designing GitHub command line products.

Learn about CLI and Primer

### [GitHub a11y checker](https://github.com/khiga8/github-a11y)

A browser extension that encourages accessibility awareness of headings and images while using GitHub.

Get the extension
