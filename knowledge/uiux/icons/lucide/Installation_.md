# Installation [​](#installation)

## Web [​](#web)

Implementation of the lucide icon library for web applications.

pnpmyarnnpmbun

sh

```
pnpm add lucide
```

sh

```
yarn add lucide
```

sh

```
npm install lucide
```

sh

```
bun add lucide
```

For more details, see the [documentation](./packages/lucide).
